\section{Piar correlation function using MC simulations in the NVT ensemble}
\subsection{}
The pair corellation function is given by:
\begin{align*}
    g(r)= \frac{1}{\rho^2} \left\langle \sum_{i =1}^{N} \sum_{j \neq i}^{N} \delta(\mathbf{r_i})\delta(\mathbf{r_j - r})\right\rangle = \frac{V}{N^2} \left\langle \sum_{i =1}^{N} \sum_{j \neq i}^{N} \delta(\mathbf{r - r_{ij}})\right\rangle
\end{align*}

For large distances the postions of particle i and j become uncorelated. This means that the pair distribution function can be written as an product of two one-particle distributions: $\rho^{(2)}(\mathbf{r,r^{'}}) = \rho^2 g(r) \rightarrow \rho^{(1)}(\mathbf{r})\rho^{(1)}(\mathbf{r^{'}})$ For our pair correlation function this means:
\begin{align*}
    g(r) \rightarrow \frac{1}{\rho^2} \left\langle \sum_{i =1}^{N}\delta(\mathbf{r-r_i})\right\rangle \left\langle \sum_{j =1}^{N}\delta(\mathbf{r-r_j})\right\rangle = \frac{1}{\rho^2} \rho \rho = 1
\end{align*}
\subsection{}
We start of with a FCC latice from exercise 4. 
We than use the Monte Carlo simulation of the NVT ensemble from exercise 4 again to equilibriate the system for 10.000 steps. 
After the system is in equilibrium we run the simulation for another 10.000 steps where we take a snapshot of the coordinates every 1.000 steps. 
For every snapshot of the coordinates we calcualte the pair correlation function according to formula 8.2 from the exercise:
\begin{align*}
    g(r + 0.5\delta r) = \frac{1}{N} \frac{n_{his}(b)}{n_{id}(b)}
\end{align*}
Here $n_{his}$ is the amount of pairs in the interval $r, r + \delta r$ and $n_{id}$ is the theoretical count for the ideal gass: $n_{id} = \left(\frac{4 \pi N}{V} \right)\left[\left(r + dr \right)^3 -r^3 \right]$.
The code is in appendix \ref{app: Ex6}.

Here we have 6 plots of the radial distribution function for different packing fractions ranging from 0.2 to 0.45.
For a $r/\sigma$ lower than 1 the radial distribution is zero across all packing fractions. This is exactly what we expect for our HS potenial where there are no other partickles within one diameter.
We get peaks at integer values of $r/\sigma$ and minima in between our peaks at half integer values of  $r/\sigma$.
